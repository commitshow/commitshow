# Team photos

Drop founder portraits here:
- `han.jpg` — Han Kim
- `cj.jpg`  — CJ Kim

Both pitch pages (`/pitch` and `/pitch-k`) load these directly. If a
file is missing, the FounderCard component falls back to a single-
letter gold tile (initial-based avatar) — no broken-image icon, the
deck always looks clean.

Recommended specs:
- 1:1 square, 256–512 px each side (we render at 64×64)
- Cropped tight to face / shoulders
- Either solid background or natural backdrop · the deck's navy
  surround tolerates both
